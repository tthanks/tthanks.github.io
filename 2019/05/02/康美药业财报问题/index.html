说没就没了

发表于 2019-05-02 | 分类于 雪球

2019年4月30日，康美药业披露对前期会计差错进行更正，称公司经过自查，对2017年度财务报表进行重述，涉及多项报表科目。其中，分别调减货币资金营业收入，营业成本299.44亿元、88.98亿元、76.62亿元等，调增存货、其他应收款、应收账款、在建工程金额为195.46亿元、57.14亿元、6.41亿元和6.32亿元等。（上交所公告）

简单来说，就是说了一个事情，2017年货币资金多计算了299.4亿元。从披露来看，财务问题已经由来已久。很早被人诟病的公司资金以及经营性现金流状况不匹配。一个正常的企业，是靠出售其产品获取利润，而正常的现金流增加，也应该是这样的。

当日公司的股票也毫无悬念的跌停了。作为投资路上的重要教训，值得记录。

任何情况下，单个资产所持有的比例不宜过高。
